# LMS-project
# StarClass AI — Learning Management System

A full-stack Learning Management System built with React, Node.js, and MySQL.
Students learn from AI-powered celebrity mentors across tech courses.

## 🚀 Tech Stack

**Frontend:** React + Vite + Tailwind CSS  
**Backend:** Node.js + Express  
**Database:** MySQL (Sequelize ORM)

## 📁 Project Structure
LMS-project/

├── backend/          # Node.js + Express API

├── frontend-student/ # Student portal (React)

└── frontend-admin/   # Admin dashboard (React)

## ⚙️ Setup & Installation

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend - Student
```bash
cd frontend-student
npm install
npm run dev
```

### Frontend - Admin
```bash
cd frontend-admin
npm install
npm run dev
```
